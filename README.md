# pizza-sales-dashboard
Pizza Sales Dashboard in Power BI

## Project Overview
This project analyzes pizza sales performance using an interactive Power BI dashboard. It provides insights into revenue, product performance, and sales trends across different locations.

## Business Objective
-Understand sales performance by pizza type and category.

-Identify top selling products

-Analyze revenue trends over time

-Support data driven decision making

## Key Metrics | KPI
-Number of Pizzas Sold: 211 (Harbor Front)

-Total Revenue: 3.58K (Harbor Front)

-Top Pizza Flavors:

-Pepperoni

-Margherita

-BBQ Chicken

## Key Insights
-Pepperoni and Margherita are the top-performing pizza flavors

-Normal crust dominates customer preference (74% Downtown Plaza)

-Tomato based sauces are the most popular (39% Downtown Plaza)

-Sales show fluctuations with peaks around mid-period

-Certain flavors (e.g., Four Cheese, Mediterranean) underperform and may need promotion
